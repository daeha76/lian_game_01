# 리안이의 요리 게임 🍰

초등학교 1학년 딸이 만들고 싶어한 요리 게임이에요. 단계별 안내를 따라가며 음식을 만들어요.

## 플레이 방법

가장 쉬운 방법: [`index.html`](./index.html) 파일을 더블클릭하면 브라우저에서 열려요.

로컬 서버로 띄우려면 (선택):

```bash
python -m http.server 8765
```

그 다음 브라우저에서 `http://localhost:8765` 접속.

PC, 태블릿, 모바일 모두 지원해요.

## 현재 레시피

- **딸기 빵** 🍓 — 핑크색 음식 (10단계)

## 게임 흐름

1. 시작 버튼
2. 재료 모으기 (밀가루, 우유, 딸기, 달걀)
3. 반죽 조물조물
4. 반죽 밀기
5. 빵을 가스레인지에 넣기
6. 굽기 🔥
7. 뚜껑 열기
8. 빵 꺼내기
9. 딸기잼 바르기
10. 완성! ✨

## 새 레시피 추가하기

[`index.html`](./index.html) 안의 `RECIPES` 객체에 새 음식을 한 덩어리로 추가하면 돼요.

## 기술

- HTML / CSS / JavaScript (단일 파일)
- 외부 라이브러리 없음
- 캔버스(Canvas)로 잼 바르기 구현
